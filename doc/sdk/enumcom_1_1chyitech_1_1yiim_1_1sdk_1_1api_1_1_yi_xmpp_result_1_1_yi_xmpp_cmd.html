Detailed Description

用于唯一标识一个Xmpp操作，如调用YiIMSDK.getInstance().connect(listener)时 连接成功或失败后，会通过类型为YiXmppListener的listener进行操作结果回调 void onXmppResonpse(YiXmppResult result); 其中result.what即为标识此操作结果为哪一个操作的结果。
